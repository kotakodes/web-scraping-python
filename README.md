# Web Scraping with Python (Requests + BeautifulSoup)

This project is a beginner-friendly introduction to web scraping using Python.

## 📌 What This Project Covers

* Sending HTTP requests using the Requests library
* Parsing HTML using BeautifulSoup
* Extracting structured data from web pages
* Navigating HTML tags and attributes
* Using loops to scrape multiple pages
* Cleaning data using regular expressions

## 🌐 Website Used

http://books.toscrape.com/
(This is a practice website created specifically for web scraping.)

## 🛠️ Technologies Used

* Python
* Requests
* BeautifulSoup

## 🚀 How to Run

1. Clone this repository
2. Install dependencies:
   pip install requests beautifulsoup4
3. Run the script:
   python web_scraping_intro.py

## 📂 Output

The script extracts:

* Book titles
* Prices
* Availability status
